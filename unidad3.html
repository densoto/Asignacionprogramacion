Objetos

La unidad basica de la programacion orientada a objetos es el "objeto". Los objetos son entidades que tienen un determinado "estado", "comportamiento (metodo)" e "identidad".

Agregar Contenido.

Un objeto es una combinacion de codigo y datos que puede tratarse como una unidad, puede ser una parte de una aplicacion, como un control, un formulario o toda la aplicacion.

Cuando se crea una aplicacion en Visual Basic, se trabaja constantemente con objetos como los proporcionados por Visual Basic (controles, formularios y objetos de acceso a datos); de igual forma, podemos crear nuestros propios objetos y definir propiedades y metodos adicionales para ellos. Los objetos actuan como bloques de compilacion prefabricados que permiten escribir un fragmento de codigo de una vez y utilizarlo una y otra vez.

Crystal Report

Es una aplicacion que permite crear informes con diseños atractivos. Puede estraer la informacion de una amplia gama de fuentes de datos.

Agregar Contenido.

Es una aplicacion que permite crear informes con diseños atractivos. Puede estraer la informacion de una amplia gama de fuentes de datos.

En Visual Studio 2017 no se instala de forma predeterminada, es necesario descargar Crystal Report para Visual Studio 2017 e instalarlo. Los datos para un reporte pueden provenir de una sola tabla o de multiples tablas. MySQL permite crear Vistas, las cuales pueden ser referenciadas con instrucciones select como si se trataria de una tabla mas.
